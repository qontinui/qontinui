Integration with QontinUI
-------------------------
The parser is integrated into the static extraction module:

from qontinui.extraction.static import TypeScriptParser

This allows it to work alongside:
- ReactStaticAnalyzer
- NextJSStaticAnalyzer
- Other static analyzers

The parser provides data for:
- State modeling
- Transition discovery
- UI structure analysis
- Test generation
- Documentation generation
